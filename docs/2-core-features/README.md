# Core Features

This section deep-dives into the standard features of the Xylem engine. Click on any section to see technical details and API references:

*   [MVCC & Transactions](mvcc-and-transactions.md) — Isolated writing, transactional locking, conflict resolution, and assertions.
*   [Graph Traversal](graph-traversal.md) — Custom graph pathing query clauses to read/write complex path networks.
*   [Vector Search (HNSW)](vector-search-hnsw.md) — High performance vector indexing, on-demand ingestion, and similarity pruning.
*   [Blob Store & CAS](blob-store-and-cas.md) — Blake2b hashing, deduplication, transparent blob columns, reference counts, and partial updates.
*   [Reactivity & Pub/Sub](pubsub-and-virtual-columns.md) — Active row watching, callbacks, and virtual column communication.
*   [Security & Encryption](security-and-encryption.md) — Multi-key on-the-fly encryption/decryption of rows.
